# JS-shake
一个简单的震动效果

===============================

1. // 实例化
2. var shake = new Shake('shake-id');
3. // 开始震动
4. shake.start();
5. // 停止震动
6. shake.stop();

===============================

 * el 震动的元素对象或者元素ID('shake-id') el需设置相对或者绝对定位
 * opts.margin 震动幅度(上/左切换偏移距离) 默认 2(px)
 * opts.rate 震动切换间隔时间 (上/左切换) 默认 20(ms) 间隔时间*2不能大于重复时间
 * opts.delay 震动重复时间 频率(重复上/左切换)  默认 60(ms)
 * ====================================================
 * this.start() 开始震动
 * this.stop() 停止震动
